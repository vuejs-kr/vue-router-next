## 빠른시작

- CDN을 통해: `<script src="https://unpkg.com/vue-router@4"></script>`
- [CodeSandbox](https://codesandbox.io/s/vue-router-4-reproduction-hb9lh)의 브라우저 내 playground
- 기존 Vue 프로젝트에 추가:
  ```bash
  npm install vue-router@4
  ```
